# Jquerycsv::Rails

Rails gem tracking jquery csv

## Installation

Add this line to your application's Gemfile:

    gem 'jquerycsv-rails'

And then execute:

    $ bundle

Or install it yourself as:

    $ gem install jquerycsv-rails
